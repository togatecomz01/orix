Button Standard

1. 버튼 유형 및 용도

  • .btn-primary

    메인 액션(확인, 다음, 제출). 브랜드 컬러 적용.

  • .btn-secondary

    서브 액션(취소, 이전). 흰색 배경 + 테두리.

  • .btn-outline

    컨텐츠 내부 보조 버튼(PDF 다운로드 등).

  • .disabled

    유효성 미통과 시 비활성 상태 표시.

2. 레이아웃 규칙

  • 하단 버튼

    #footerWrap > .btn-area 내부에 배치.

  • 컨텐츠 버튼

    .acc-btn-area 내부에 배치 (우측 정렬).

  • 팝업 버튼

    .pop-footer 내부에 배치 (중앙 정렬).
